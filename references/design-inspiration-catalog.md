## 如何使用此目录

1. **Phase 2 扫描前**：根据当前 artifact_type 和目标领域，从上面对应分类中选取 2-3 个参考公司
2. **Design Best-Practice Scan**：将选中公司的视觉特征作为 Enterprise Product Patterns 层的具体来源
3. **Adopt / Reject**：记录哪条设计决策受到了哪家公司的启发，写明原因
4. **Web search 兜底**：如果目录中没有覆盖到的领域，使用 web search 搜索 "best <domain> website design" 或 "<company> design system"
5. **扩展搜索种子**：[awesome-design-systems](https://github.com/alexpate/awesome-design-systems)（24K+ stars）索引了 200+ 真实设计系统。当目录覆盖不足时，以此仓库的公司列表作为 web search 种子，搜索 `<company-name> design system` 获取具体设计 token 和视觉规范
