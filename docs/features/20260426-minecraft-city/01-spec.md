# Matrix Hub — Minecraft 现代城市中心规划

## 项目概述

创造模式大项目：200×200 模块化城市中心区。四个角塔 + 玻璃穹顶交通枢纽 + 架空空中走廊 + 地面公园广场 + 地下服务层。

---

## 核心尺寸

| 元素 | 尺寸 | 说明 |
|------|------|------|
| 基础平台 | 200 × 200 | 平坦方形区域，Y 坐标先定基准面 |
| 中央穹顶 | Ø 60 | 玻璃穹顶，覆盖交通枢纽 |
| 角塔（4 座） | 20×20 基底，高 150+ | 分别对应办公/酒店/住宅/观景 |
| 空中走廊 | 宽 5 | 连接 4 座塔楼与中央穹顶 |
| 地面层高 | 4 (Y+5) | 地面广场层净高 |
| 地下层 | 1~2 层 | 服务层（商店/机房/通道） |
| 街区道路 | 宽 7~9 | 主干道 + 次干道 |

## 材料色板

| 用途 | 方块 | 备注 |
|------|------|------|
| 塔楼外墙 | 白色混凝土、浅灰混凝土、石英块 | 现代立面 |
| 塔楼幕墙 | 浅蓝/浅灰染色玻璃 | 镜像效果 |
| 塔楼框架 | 铁块、石砖 | 结构线条 |
| 中央穹顶 | 玻璃块（浅蓝/白染色） + 海晶灯 | 发光穹顶 |
| 地面广场 | 石砖、混凝土粉末（灰/白）、草径 | 纹理混铺 |
| 空中走廊 | 玻璃板 + 铁活板门框架 | 透明步道 |
| 地下层 | 石砖、海晶灯、彩色混凝土 | 照明充足 |
| 装饰 | 海晶灯、灯笼、花盆、树叶 | 点缀用 |

## 色系方案

- **主色**: 白 + 浅灰（建筑主体）
- **辅色**: 深灰 + 铁色（结构线条）
- **点缀**: 浅蓝（玻璃）+ 绿植（屋顶花园）
- **夜景**: 海晶灯 + 橙黄灯笼 = 暖冷对比

## 分区规划

```
        ┌──────────────────────────────────────┐
        │  角塔 A          │         角塔 B     │
        │  (办公/商务)     │      (酒店/商业)    │
        │                   │                    │
        │    ─── 空中走廊 ──── 空中走廊 ───      │
        │                   │                    │
        │         ╔═══════════════╗              │
        │         ║  中央穹顶     ║              │
        │    ───  ║  (交通枢纽)   ║  ───         │
        │         ╚═══════════════╝              │
        │                   │                    │
        │    ─── 空中走廊 ──── 空中走廊 ───      │
        │                   │                    │
        │  角塔 C          │         角塔 D     │
        │  (住宅/公寓)     │      (观景/文化)    │
        └──────────────────────────────────────┘
```

### 地面层（Y+0）

- 四大广场分区，各自有不同的铺装纹理
- 中央穹顶周围环形步行道
- 绿植带、小型喷泉、休息区
- 道路从城市外围延伸接入

### 空中走廊层（Y+15~20）

- 4 条走廊，每条连接一个角塔到中央穹顶
- 走廊为全玻璃封闭通道，宽 5
- 角塔之间不做直接连接（保持视线通透）

### 角塔功能分配

**角塔 A — 办公/商务（东南）**
- 底层大堂：高 10 层高
- 中层：标准办公层（5 层，每层 shell）
- 顶层：观景层 + 露台
- 外墙：白色混凝土 + 纵向玻璃条

**角塔 B — 酒店/商业（西南）**
- 底层：商业街区（小店铺 facade）
- 中层：酒店客房层（shell）
- 顶层：空中大堂/酒吧（装修）
- 外墙：浅灰混凝土 + 横向玻璃带

**角塔 C — 住宅/公寓（西北）**
- 底层：社区大厅
- 中层：住宅层（阳台外挑）
- 顶层：屋顶花园 + 泳池（装饰）
- 外墙：白色+深灰条纹，阳台绿植

**角塔 D — 观景/文化（东北）**
- 底层：展览馆（开放空间）
- 中层：空（留白，或做室内绿化）
- 顶层：360°观景台（装修）
- 外墙：全玻璃幕墙为主

### 中央穹顶 — 交通枢纽

- 半球形或椭球穹顶，底直径 ~50，高 ~30
- 内部：地铁站厅设计（即使无实际红石）
- 四条走廊入口在穹顶四方向
- 中央：下沉式售票大厅 + 电子屏装饰
- 顶部：海晶灯阵列，夜晚发光

### 地下层

- **B1**: 商业街 + 停车场 + 机房
  - 中央大道连接四个角塔的地下入口
  - 小型商店 facade 沿街排列
  - 照明：海晶灯嵌入天花板
- **B2**（可选）: 预留，或做更深的机械层

## 建造阶段

### Phase 1: 场地准备
- 平整 200×200 平台
- 标注四角及中心点坐标
- 铺设基础路面

### Phase 2: 核心结构
- 中央穹顶骨架 + 玻璃填充
- 四座角塔的框架（先做结构，不做细节）
- 空中走廊骨架

### Phase 3: 地面层
- 广场铺装 + 绿植 + 装饰
- 外围道路衔接
- 路灯/照明系统

### Phase 4: 建筑细化
- 角塔外墙 + 幕墙填充
- 各塔顶层观景台装修
- 大堂入口设计

### Phase 5: 穹顶内饰
- 内部站厅设计
- 照明阵列
- 四个入口大厅

### Phase 6: 地下层
- B1 开挖 + 装饰
- 连通四个角塔的地下通道

### Phase 7: 收尾
- 空中走廊内部装饰
- 夜景照明（海晶灯 + 红石灯）
- 周边环境（衔接地形、绿化）
- 截图记录

## 关键技术指标

- 四座角塔高度在 150~180 之间不等（避免完全对称）
- 中央穹顶最高点 ~Y+40 (从地面算起)
- 空中走廊在 Y+20 层连接
- 地下层深度 Y-5~Y-8
- 平台基准面建议选海洋/河流平面或超平坦世界
