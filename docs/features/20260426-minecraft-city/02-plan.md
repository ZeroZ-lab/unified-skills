# Plan: Matrix Hub — 建造任务分解

> 基于: `docs/features/20260426-minecraft-city/01-spec.md`

## 坐标约定

- 基准坐标: `0 0` 为平台中心
- 平台范围: X -100~100, Z -100~100
- 基准面 Y = 64（超平坦世界海平面）
- 地面层: Y=64
- 空中走廊: Y=80~85
- 地下层: Y=56~60
- 角塔高度: Y=64~220 (156 层)

```
       角塔 C (-60,-60)           角塔 B (60,-60)
             西北                       西南
                  \                    /
                   \  中央穹顶 (0,0)  /
                   /                  \
                  /                    \
       角塔 D (-60,60)            角塔 A (60,60)
             东北                       东南
```

---

## Phase 1: 场地准备

**目标:** 平整 200×200 平台，划分分区

| 步骤 | 操作 | WorldEdit 命令 |
|------|------|----------------|
| 1.1 | 平整平台 | `//pos1 -100,63,-100` → `//pos2 100,63,100` → `//set grass_block` |
| 1.2 | 标记中心 | 在 (0,64,0) 放置一块金块 |
| 1.3 | 标记四角 | 在 (±60,64,±60) 放置四块不同羊毛（A=红, B=蓝, C=绿, D=黄） |
| 1.4 | 铺设主干道 | 从角到中心，宽 7，用 stone_brick 铺设十字路 |

## Phase 2: 核心结构

### 2.1 中央穹顶骨架

```
位置: 中心 (0,0)
结构: 椭球穹顶
```

- 基底直径 50，高 30
- 用 glass (light_blue_stained_glass) 加 iron_bars 框架
- WorldEdit: `//hsphere glass 25` 做胚体 → 手动修成半椭球

### 2.2 四座角塔框架

使用相同基底 20×20，高度各不相同避免完全对称：

| 角塔 | 坐标 (中心) | 高度 (Y) | 总层数 | 
|------|------------|----------|--------|
| A (办公) | (60, 64, 60) | 64~200 | 34 (每层 4) |
| B (酒店) | (-60, 64, 60) | 64~180 | 29 (每层 4) |
| C (住宅) | (-60, 64, -60) | 64~160 | 24 (每层 4) |
| D (观景) | (60, 64, -60) | 64~220 | 39 (每层 4) |

标准层模板（20×20 基底）：

```
  ■■■■■■■■■■■■■■■■■■■■
  ■                  ■
  ■  白色混凝土外墙   ■
  ■  (玻璃幕墙窗口)   ■
  ■  铁块结构柱       ■
  ■                  ■
  ■■■■■■■■■■■■■■■■■■■■
```

- 四角立柱用 iron_block
- 外墙用 white_concrete
- 窗口用 light_blue_stained_glass

WorldEdit 技巧：建一层 → `//copy` → `//stack` 重复堆叠

### 2.3 空中走廊骨架

4 条走廊，每条从角塔内侧连接到中心穹顶：

| 走廊 | 连接 | 长度 | Y 高度 |
|------|------|------|--------|
| A→中心 | (60,64,60)→(0,64,0) 沿 Z 轴 | 60 | Y=80~85 |
| B→中心 | (-60,64,60)→(0,64,0) 沿 X+Z | ~85 | Y=80~85 |
| C→中心 | (-60,64,-60)→(0,64,0) 沿 Z | 60 | Y=80~85 |
| D→中心 | (60,64,-60)→(0,64,0) 沿 X | ~85 | Y=80~85 |

走廊截面：宽 5 × 高 4：
```
 ■■■■■■■
 ■     ■
 ■   □  ■  ← 铁块框架 + 玻璃底板
 ■     ■
 ■■■■■■■
```

## Phase 3: 地面层

### 3.1 广场铺装

分区纹理混铺：

| 区域 | 主材 | 辅材 |
|------|------|------|
| 中央环岛 | white_concrete | 海晶灯做环形图案 |
| 东南 (A区) | stone_brick | 石台阶 |
| 西南 (B区) | 浅灰混凝土粉末 | 灰混凝土粉末 |
| 西北 (C区) | grass_block + 草径 | 树叶 |
| 东北 (D区) | 深色橡木 | 灯笼 |

### 3.2 道路系统

- 四向主干道从城市外围 → 中心
- 宽 7：中间 3 灰混凝土 + 两边各 2 深灰混凝土
- 路段每 10 格一盏灯笼

### 3.3 绿化

- 中央环岛外围一圈树叶（oak_leaves）
- 四区之间用树篱分隔
- 小型花坛点缀

---

## Phase 4: 建筑细化

### 4.1 塔楼外立面

逐个为角塔做外墙：白色混凝土 + 玻璃幕墙间隔。

标准立面节奏（每层）：

```
 ■■■□□□□□□□□□□□□□□□□□■■■
 ■  ■                 ■  ■
 ■  ■                 ■  ■
 ■■■□□□□□□□□□□□□□□□□□■■■
```

- ■ = white_concrete 结构柱
- □ = glass 窗口
- 外墙交接处用 iron_block

### 4.2 塔楼大堂

每个角塔底部 3 层打通为大堂层高：

- 地面：polished_blackstone 或 dark prismarine
- 前台：用 楼梯 + 按钮 做装饰
- 灯光：海晶灯吊顶

### 4.3 顶层观景台

**角塔 D 重点装修**（全玻璃顶层）：

- 四周全 glass 墙，视野通透
- 地面：海晶灯 + 浅蓝地毯
- 栏杆：iron_bars
- 可放置若干 item_frame 做装饰

---

## Phase 5: 穹顶内饰

### 5.1 穹顶内部

- 内部架空，不设楼层
- 底直径 50 → 内部空间约 50×50
- 地面：polished_andesite + 海晶灯十字图案

### 5.2 交通枢纽设计

- 中央下沉式区域（凹陷 3 层）
- 四条通道通向四个入口
- 入口处拱门用 quartz_block + 海晶灯
- 悬挂式"时刻表"（item_frame + map）

### 5.3 灯光

- 穹顶顶点放 sea_lantern 集群
- 内壁每隔 5 格嵌入 sea_lantern
- 夜间由内向外透光

---

## Phase 6: 地下层

### 6.1 B1 开挖

`//pos1 -95,56,-95` → `//pos2 95,60,95` → `//set stone`

### 6.2 B1 装饰

- 天花板：sea_lantern 每 7×7 一个
- 中央大道宽 7，和地面层对齐
- 两侧小型"店铺"（门面 5×5 凹陷 + 不同色块）
- 楼梯/电梯井在四角塔下方

### 6.3 垂直交通

- 每个角塔下方设楼梯井：4 格宽螺旋梯
- 中央穹顶下方设主楼梯到 B1

## Phase 7: 收尾

### 7.1 周边过渡

- 平台边缘用 grass 台阶自然下沉
- 向外延伸 10 格绿化带
- 外围道路衔接

### 7.2 截图

- 日景全景（从东南高空俯瞰）
- 夜景全景（展示海晶灯效果）
- 各角塔特写 + 穹顶内部
- 空中走廊视角

```
Phase 1 (场地)
   │

   │                        │
   ▼                        ▼
Phase 3 (地面) ─────→ Phase 6 (地下层)
   │
   ▼
```
