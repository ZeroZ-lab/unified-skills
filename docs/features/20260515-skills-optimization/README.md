## 优化模式总结

每个技能统一新增以下结构模式：

1. **前置加载 + 输出路径** — 替换旧术语 "假设已加载"，声明 CANON.md + 下游技能
2. **说辞+现实+可量化后果** — 三列表格，后果列用量化成本（时间/金钱/用户影响）
3. **验证失败处理表** — 失败场景 | 处理方式，≥4 行覆盖核心失败
4. **好坏示例** — Good + Bad 对比，描述具体行为和后果
5. **输出模板** — markdown code block 展示产出格式
6. **Step Checkpoints** — 低分技能额外添加流程检查点
